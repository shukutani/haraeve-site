## 定時退社（カードゲーム）

<p style="font-size:1.1rem; font-weight:600; color:#16a34a; margin-bottom:1.5rem;">
仕事を片付けて、誰よりも早く退社しろ！
</p>

<p style="margin-bottom:2rem;">
ババ抜きをベースにした、<br />
<span style="color:#0284c7; font-weight:600;">
短時間・大盛り上がりの正体隠匿系カードゲーム
</span>です。
</p>

<hr style="margin:2.5rem 0;" />

<h3 style="font-size:1.5rem; font-weight:700; margin-bottom:1.2rem;">
基本情報
</h3>

<ul style="margin-bottom:2.5rem;">
  <li style="margin-bottom:0.6rem;"><strong>プレイ人数：</strong>3〜6人</li>
  <li><strong>プレイ時間：</strong>3〜10分</li>
</ul>

<hr style="margin:2.5rem 0;" />

<h3 style="font-size:1.5rem; font-weight:700; margin-bottom:1.2rem;">
ゲームの目的
</h3>

<p style="margin-bottom:1.2rem;">
同じ名前の<strong>「仕事カード」</strong>を2枚揃えて場に出し、<br />
<span style="font-weight:600; color:#f59e0b;">
いち早く手札を0枚にして退社すること
</span>が目的です。
</p>

<p style="margin-bottom:1.2rem;">
最後まで
<span style="color:#dc2626; font-weight:600;">
残業（ババ）
</span>
を持っていた人は……
</p>

<p style="font-weight:700; margin-bottom:2.5rem;">
残念、今日も残業です。
</p>

<hr style="margin:2.5rem 0;" />

<h3 style="font-size:1.5rem; font-weight:700; margin-bottom:1.2rem;">
ゲームの流れ
</h3>

<ol style="margin-bottom:2.5rem;">
  <li style="margin-bottom:0.8rem;">右隣の人から、裏向きのカードを1枚引く</li>
  <li style="margin-bottom:0.8rem;">同じ仕事カードが2枚揃ったら、場に出して捨てる</li>
  <li style="margin-bottom:0.8rem;">仕事カードの効果を使う（使わなくてもOK）</li>
  <li>
    <span style="font-weight:600; color:#16a34a;">
      手札が0枚になったら退社（勝ち抜け）
    </span>
  </li>
</ol>

<hr style="margin:2.5rem 0;" />

<h3 style="font-size:1.5rem; font-weight:700; margin-bottom:1.2rem;">
ゲームの終了
</h3>

<ul style="margin-bottom:2.5rem;">
  <li style="margin-bottom:0.6rem;">残業カードを持っている人が1人になったらゲーム終了</li>
  <li>
    <span style="color:#dc2626; font-weight:600;">
      その人が敗者（残業）
    </span>
  </li>
</ul>

<hr style="margin:2.5rem 0;" />

<h3 style="font-size:1.5rem; font-weight:700; margin-bottom:1.2rem;">
盛り上げポイント
</h3>

<ul style="margin-bottom:2.5rem;">
  <li style="margin-bottom:0.6rem;">1ゲームが短く、初対面でも遊びやすい</li>
  <li style="margin-bottom:0.6rem;">社会人ネタ満載で自然と会話が生まれる</li>
  <li>
    <span style="color:#0284c7;">
      カードを出すときにセリフを言うとさらに楽しい
    </span>
  </li>
</ul>

<hr style="margin:2.5rem 0;" />

<h3 style="font-size:1.5rem; font-weight:700; margin-bottom:1.2rem;">
こんな人におすすめ
</h3>

<ul>
  <li style="margin-bottom:0.6rem;">軽く遊びたい人</li>
  <li style="margin-bottom:0.6rem;">初対面の人とすぐ仲良くなりたい人</li>
  <li>
    <strong>定時退社したいすべての社会人</strong>
  </li>
</ul>
